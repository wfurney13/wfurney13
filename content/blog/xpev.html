Xonsh is a "Python-powered, cross-platform, Unix-gazing shell language and command prompt." 1 Many operations that are clumsy in Windows natively are comparatively simple using the Xonsh shell. Let's take a look at one example, modifying the PATH environment variable.

In Windows 11 without Xonsh, modifying your PATH environment variable can be done in either the "Environment Variables" window or by updating this key in the Registry Editor:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Environment

With Xonsh, the variable is stored directly in a list of strings called $PATH. 2 Part of the magic of Xonsh is that we can use built-in Python methods against this variable! First, I need to add this line in my Xonsh configuration file:

$UPDATE_OS_ENVIRON = True

When True, the OS environment will always be updated when the xonsh environment changes. This gives us a more efficient way to interact with the PATH. 3 For example, to remove the most recent entry from the list I can use the pop() method...

$PATH.pop(-1)

Now let's say I download an application and want to store its binaries in my PATH for convenience. I can use Xonsh to call the append() method and add its path to the list:

$PATH.append('C:\\Users\\wfurn\\Speedtest-cli')

How cool is that? Other methods would work as well of course, but these two seem the most useful to me at the moment. Some great examples of pythonobash can be found in this overview video. 4
